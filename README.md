Deloitte Data Analytics Job Simulation
Project Overview

This repository contains my solutions for the Deloitte Data Analytics Virtual Job Simulation hosted on Forage.

The simulation focuses on solving real business problems using data analytics, dashboard development, and Excel-based data classification.

Business Problem

The client wanted answers to two business questions:

Task 1
Which factory experienced the highest machine downtime?
Which machine type contributed the most downtime in that factory?
Task 2
Classify employee Equality Scores into:
Fair
Unfair
Highly Discriminative

using predefined business rules.
